Este projeto consiste em um sistema simples de CRUD para cadastro e gerenciamento de produtos, desenvolvido como parte de uma entrevista técnica para uma vaga de Desenvolvedor Júnior.

A aplicação permite realizar as operações básicas de gerenciamento de dados (Create, Read, Update e Delete), possibilitando o cadastro, visualização, edição e remoção de produtos de forma prática e organizada.

O sistema foi desenvolvido utilizando Laravel no backend, responsável pela criação da API REST, regras de negócio e comunicação com o banco de dados. Já o Angular foi utilizado no frontend, proporcionando uma interface moderna e reativa para interação com o usuário.

Entre as funcionalidades implementadas estão:

- Cadastro de novos produtos

- Listagem de produtos cadastrados

- Edição de informações de produtos

- Exclusão de produtos

- Associação de produtos a categorias

- Consumo de API REST entre Angular e Laravel

O objetivo do projeto é demonstrar boas práticas de desenvolvimento, organização de código, integração entre frontend e backend e conhecimento em arquitetura básica de aplicações web, requisitos fundamentais para um desenvolvedor em início de carreira.
